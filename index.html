🌙 Kimi RECOMMENDED

How it works with Kimi (Recommended for highest quality):
1. Paste your transcript or document text below
2. Click "Generate Prompt" to create the Kimi prompt
3. Copy the generated prompt
4. Click "Open Kimi.com" and paste it there
5. Kimi will generate a quiz JSON you can paste back here

💡 Tip: For YouTube videos, get transcripts from youtubetotranscript.com
How it works with Copilot Enterprise:
1. Paste your transcript or document text below
2. Click "Generate Prompt" to create the prompt
3. Download the prompt using the "📥 Download Prompt" button (Copilot doesn't accept long paste)
4. Open Copilot Enterprise
5. Upload/attach the quiz_prompt.txt file to Copilot
6. Type a period "." or any character in the message box (required to send)
7. Copilot will generate a downloadable JSON file
8. Download and paste the JSON content back here

💡 Note: This specifically works with the Enterprise version at copilot.cloud.microsoft (which has Python code execution). The consumer Copilot (copilot.microsoft.com) does not support this workflow.
How it works with Claude or ChatGPT:
1. Paste your transcript or document text below
2. Click "Generate Prompt" to create the prompt
3. Copy the generated prompt
4. Open Claude or ChatGPT and paste it there
5. The AI will generate a quiz JSON (may be inline or as a file)
6. Copy or download the JSON and paste it back here

💡 Tip: Claude has a large context window and handles long prompts well via copy/paste. ChatGPT may vary by model.

❓ How to Use Universal Quiz Player

1

Find Your Video Transcript

Go to your learning video (YouTube, etc.) and get the transcript. For YouTube, use youtubetotranscript.com or click the "..." menu below the video and select "Show transcript".

2

Generate a Quiz Prompt

Click "🤖 Generate from Transcript", paste your transcript, and click "Generate Prompt". We recommend Kimi for the highest quality quizzes. Claude, ChatGPT, and Copilot Enterprise also work well.

3

Get Your Quiz JSON

Copy the prompt and paste it into Kimi.com (or Copilot Enterprise/Claude/ChatGPT). The AI will process it and return a quiz in JSON format.

4

Load and Take Your Quiz

Copy the JSON from the AI and paste it into the main page, then click "Load Quiz". You're ready to study! Your progress auto-saves.

💡 Pro Tip: You can also paste any valid quiz JSON directly - ask any AI to create a quiz for you following our format!

🎮 Keyboard Shortcuts

• 1-9, 0 - Select answers
• ← → or , . - Navigate questions
• S - Save progress
• Ctrl+Enter - Load quiz from JSON
